# Rainy Day

An End-to-End encrypted immutable cloud storage solution, for the rainy days.

> [!WARNING]
> This project is in heavy development

## Features

These are the features I hope to add eventually

- End-to-End Encyption so you can run the server-side on any machine without
fear of your data being accessible to anyone but you
- Non-destructive updates, previous versions of every file is always accessible
and can be rolled back
- Server-side, desktop-side and App

## Specification

This project is specified using [tracey](https://tracey.bearcove.eu/) and the
specification is in [./docs/spec/rainy_day.md]
